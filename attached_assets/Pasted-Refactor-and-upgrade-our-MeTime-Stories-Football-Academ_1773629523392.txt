Refactor and upgrade our MeTime Stories Football Academy platform into a role-based academy system.

We are no longer treating all non-parent contributors as simple external form users. We now want a more structured internal platform for academy staff.

BUILD GOAL

Create a football academy platform with two clear entry paths:

1. Player Journey

players/families complete the story journey flow

parents still receive secure links and do not need accounts

2. Academy Staff Login

internal staff log in and work from dashboards

staff can enter and review supporting material for player profiles

each academy has role-based access and a maximum of 8 staff/admin accounts

TECH STACK

Use:

React + Vite

Tailwind CSS

React Router

Supabase Auth

Supabase database

clean component structure

no giant single-file HTML or inline JS

APP STRUCTURE

Create clean folders such as:

src/pages

src/components

src/hooks

src/data

src/types

CORE PRODUCT RULES

Parents remain link-based only

Academy staff log in with email/password

Staff may still receive direct player links if needed, but should primarily work through login/dashboard

Each academy can have a maximum of 8 active staff accounts

Academy data must be isolated so staff only see their own academy’s players and submissions

Player profile becomes the central internal record

Build for future scale, not as a temporary form hack
